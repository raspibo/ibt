I'll be there!
==============

Ultra-simple booking webapp, developed for the RaspiBO makerspace.


Requirements
============

* MongoDB
* nodeJS
* npm

Installation
============

* run: npm update
* run: node app.js (or nodejs, depending on your distribution)
* connect to http://localhost:3000
* watch in awe.
